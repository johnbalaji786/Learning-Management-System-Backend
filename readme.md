# Learning Management System Backend

This repository contains the backend code for a Learning Management System (LMS) built using Node.js, Express, and MongoDB. The LMS allows users to manage courses, enrollments, and user accounts.

## Features

- User authentication and authorization using JWT
- CRUD operations for courses and enrollments
- Role-based access control (admin, instructor, student)
- RESTful API endpoints for frontend integration

## Technologies Used

- Node.js
- Express.js
- MongoDB with Mongoose
- JSON Web Tokens (JWT) for authentication
- bcrypt for password hashing

## user features
- User registration and login
- browse courses
- sedule courses
- payment processing
- lesson recordings
- feedback and ratings

## tutor features
- account creation and authentication
- lesson management
- lesson recordings
- payment and earnings tracking
- feedback and ratings

## Admin features
- user management
